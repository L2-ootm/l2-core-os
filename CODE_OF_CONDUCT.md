# Code of Conduct

## Our Commitment

We are committed to a respectful, safe, and constructive collaboration environment.

## Expected Behavior

- Be respectful and professional
- Focus on technical facts and reproducible evidence
- Give actionable feedback
- Assume good intent, request clarification when needed

## Unacceptable Behavior

- Harassment, hate speech, or personal attacks
- Doxxing or privacy violations
- Malicious contributions (backdoors, credential theft, abuse tooling)
- Repeated disruptive behavior after warnings

## Enforcement

Maintainers may remove content, reject contributions, or restrict participation for violations.

## Reporting

Report conduct concerns privately to repository maintainers via GitHub contact.

## Scope

This Code applies to repository spaces (issues, PRs, discussions, commits) and related maintainer interactions.
