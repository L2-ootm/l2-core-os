## 2) Decisões técnicas consolidadas
1. WhatsApp com **Baileys** (substitui Evolution API no plano atual).
2. Arquitetura **configurável por padrão**:
   - bootstrap por `.env`
   - overrides em `app_settings`
   - plano de wizard in-app
3. Estratégia de distribuição sem depender de Git no usuário final.
4. Estratégia **Android-first** para reduzir custo recorrente com hospedagem no início.
