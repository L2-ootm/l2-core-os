# L2 CORE OS — One Pager Comercial (MVP)

## Headline
**Infraestrutura invisível para clínicas operarem com previsibilidade, sem depender de software complexo.**

## Problema
Clínicas pequenas perdem receita por falhas operacionais recorrentes:
- confirmações e remarcações desorganizadas
- retrabalho manual em WhatsApp
- ausência de rastreabilidade e padronização

## Solução
O **L2 CORE OS** entrega uma base operacional pronta com:
- automação de fluxo WhatsApp (Baileys)
- API segura (RBAC/JWT + rate limit + webhook assinado)
- sincronização incremental para Android-first
- fallback determinístico (sem custo de IA cloud)

## Diferenciais
- **Setup local sem custo mensal obrigatório**
- **Arquitetura agnóstica** (não presa a um nicho)
- **Idempotência + reconciliação de conflitos**
- **MVP validado com GO/NO-GO automatizado**

## Prova técnica (MVP)
- 10/10 checks no script E2E
- stack containerizada e replicável
- runbooks e documentação de operação

## Modelo de adoção
1. Instalação em máquina local da clínica
2. Conexão WhatsApp via QR
3. Parametrização rápida (wizard)
4. Operação assistida + expansão para Android

## Próximos passos comerciais
- piloto em 1 clínica
- medir confirmação, no-show e tempo operacional
- transformar em case com métrica verificável
