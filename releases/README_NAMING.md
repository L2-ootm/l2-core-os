# Release Naming Convention

Padrão adotado:
`l2-core-os_<version>_<channel>_<platform>.zip`

Exemplos:
- `l2-core-os_v0.2.1_prod_windows.zip`
- `l2-core-os_v0.2.1_demo_windows.zip`
- `l2-core-os_v0.2.1_client_windows.zip`

Canais:
- `prod`: pacote principal para operação
- `client`: pacote simplificado para cliente final
- `demo`: pacote para demonstração comercial
