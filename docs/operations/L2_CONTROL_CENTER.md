# L2 Control Center (Windows)

Script unificado de operação diária:
- `l2-control.bat` (na raiz do projeto)

Atalho legado (ainda pode existir em docs antigos):
- `infra/scripts/l2-control.bat` (descontinuado)

## Recursos
1. Start seguro (up + health)
2. Stop
3. Restart seguro
4. Status consolidado (API + WhatsApp)
5. Logs rápidos (api/gateway/db)
6. GO/NO-GO
7. WhatsApp recover (reconnect + catch-up)
8. Abrir dashboard

## Como usar
1. Abra a pasta `infra/scripts`
2. Execute `l2-control.bat`
3. Use o menu numérico

## Objetivo
Reduzir erro humano e padronizar inicialização/operação segura do L2 CORE OS.
