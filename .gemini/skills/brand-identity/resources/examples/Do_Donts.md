# L2 Brand Usage: The Rules

> **Goal:** Maintain the "Dark Luxury" illusion at all costs.

## ✅ DO (The Standard)

### 1. Use the "Void" properly
*   **Do:** Use `#050505` (Void) for main backgrounds.
*   **Do:** Use `#1A1A1A` (Gunmetal) for cards/surfaces.
*   **Do:** Allow ample negative space. The machine needs to breathe.

### 2. Typography
*   **Do:** Use `Inter` for all UI text.
*   **Do:** Use `Orbitron` only for major headers (H1, H2) or data labels.
*   **Do:** Use UPPERCASE with `tracking-wider` for buttons and labels.

### 3. Motion
*   **Do:** Use `cubic-bezier(0.22, 1, 0.36, 1)`. Transitions should feel heavy and expensive.
*   **Do:** Stagger entrance animations by 50ms-100ms.

---

## ❌ DON'T (Violations)

### 1. Esthetics
*   **Don't:** Use pure black `#000000`. It kills depth.
*   **Don't:** Use drop shadows that are blurry/soft. Use sharp, directional shadows or glows.
*   **Don't:** Use rounded corners > `4px`. We are industrial, not organic.

### 2. Colors
*   **Don't:** Use random colors. If it's not in `design-tokens.json`, don't use it.
*   **Don't:** Mix "Warm" colors (Red, Orange) unless it's for a critical error. Stay Cool (Violet, Blue, Green).

### 3. Voice
*   **Don't:** Be chatty. "Here is your data!" -> "Data retrieval complete."
